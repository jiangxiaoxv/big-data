# 线程/进程
    进程：计算机中在执行的任务  --- 在cpu上执行和计算
    线程：进程中的小任务。
    注意：一个核上往往只能执行一个进程中的一个线程。--- 计算机看起来是在运行多个进程，实际上
        是因为在计算机中任务切换速度非常快，超过人的反应。--- 进程的执行在宏观上并并行的，在微观上
        是串行的。
