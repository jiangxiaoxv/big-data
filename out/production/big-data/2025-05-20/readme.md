# 套接字
    - 进行网络数据传输的一套API --- 本质上是可以在网络上使用流
# 网络基本概念
    - 七层模型： 物理层 数据链路层 网络层 
    - 传输层 （UDP/TCP） 会话层 应用层 
    - 表示层 HTTP、FTP、SMTP

# IP地址：在网络中标记主机。 IPV4 ---- 四组数表示一个IP地址10.8.33.3
    - 每一组数的取值是0-255  10.8.33.5
    - ipv6 是六组数表示一个地址
    - ipV9

# UDP
    1. 基于流的，不可靠的,不建立连接，需要对数据进行封包
    2. 每个包不超过64K大小
    3. 传输速度相对比较快
    4. 适用于对速度依赖性比较强，但是对可靠性依赖比较低的场景。
